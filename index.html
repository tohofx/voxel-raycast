3D Voxel Ray Casting

The canvas on this page shows a voxel grid rendered in real time using ray casting

This implementation uses a digital differential analyzer to traverse the grid.

Since the rays are currently simulated on the CPU the resolution is at 160 by 90 rays.

The camera can be moved with W, A, S, D, while yaw and pitch can be changed with the arrow keys.

To improve performance the ray casting will be moved to the GPU to leverage its parallel computing capabilities.
There will also be a migration to WebGPU once the WebGL2 prototype works to make use of the features coming with the new API.
